Responsive Calculator with Dark and Light Mode

Project Overview
This project is a responsive calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations through an interactive and user-friendly interface. The calculator includes Dark Mode and Light Mode switching, keyboard support, and responsive design for different screen sizes.

Features
• Addition, Subtraction, Multiplication, and Division
• Dark Mode and Light Mode Toggle
• Keyboard Input Support
• Clear and Delete Functions
• Responsive Design
• CSS Grid Layout
• Smooth Hover Animations

Technologies Used
• HTML5
• CSS3
• JavaScript

Learning Outcomes
• Developed a responsive web application using HTML, CSS, and JavaScript.
• Implemented calculator functionality using JavaScript functions and event listeners.
• Applied CSS Grid for button alignment and layout design.
• Added theme-switching functionality with Dark and Light modes.
• Enhanced user experience with animations and keyboard support.
